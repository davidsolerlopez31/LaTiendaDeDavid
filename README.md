# 🛒 La Tienda de David

Marketplace online creado con HTML, CSS y JavaScript.

## 📌 Descripción

La Tienda de David es una tienda web moderna donde se pueden mostrar productos de diferentes categorías como moda, tecnología y accesorios.

El proyecto está diseñado para funcionar en ordenadores, tablets y móviles.

---

## ✨ Características

✅ Diseño moderno tipo marketplace  
✅ Página responsive para móvil  
✅ Buscador de productos  
✅ Categorías de productos  
✅ Tarjetas de productos con imágenes  
✅ Sistema básico de carrito  
✅ Animaciones y efectos visuales  
✅ Código organizado y fácil de modificar
